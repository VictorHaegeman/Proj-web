A propos de ce site

Ce site vitrine est un projet de programmation Web,
réalisé par des étudiants ING1 du département d'Informatique de l'EFREI.

Contexte du projet

Ce projet a pour objectif de créer la partie vitrine institutionnelle du site du département d'Informatique de l'EFREI. Il permet de présenter l'école, les formations, l'équipe enseignante et les admissions de manière moderne et attractive.

React & TypeScript

Tailwind CSS

Design responsive

Validation et formulaires

Animation CSS

Architecture modulaire
